# TASKS
1. **Hero Section multilanguage**
   • File target: `templates/index.html`  
   • Inserisci subito dopo `<header>` un blocco Jinja `{% trans %}` con:
     - `h1` ≤ 60 char:  
       - IT → “Contenuti pronti, zero stress.”  
       - EN → “Content done. Zero stress.”  
       - ES → “Contenido listo. Sin estrés.”  
     - `p` ≤ 120 char che spiega AI trilingue & localizzazione.  
     - 3 bullet con emoji ☑️, ciascuno preso da una USP diversa.  
     - CTA `<a>` class `btn-primary` → “Prova gratis”.
   • Mobile-first, poi breakpoint `md:`.

2. **Section “Perché unico” (8 USP)**
   • Sotto l’hero aggiungi una `<section id="unique">` grid-3 colonne `lg:grid-cols-4`.
   • Ogni card: icona SVG, titolo USP, descrizione max 40 parole.
   • Cicla su lista USP in un array Jinja per facilità di traduzione.

3. **Localizzazione**
   • Usa gettext: aggiorna/crea `translations/it/LC_MESSAGES/messages.po`,
     `en`, `es`.  
   • Aggiungi nuove stringhe con `pybabel extract` e `pybabel update`.

4. **SEO & JSON-LD**
   • In `<head>` di `base.html` inserisci blocco Jinja che compila:
     - `title`, `meta name="description"` multilingua.  
     - `<script type="application/ld+json">` con `@type:"SoftwareApplication"`,
       `applicationCategory:"Marketing"`, `operatingSystem:"Web"`.
   • Chromatic lint con `npm run prettier` se presente.

5. **Stub Tracking UTM & Predictive CTR**
   • Crea in `static/js/analytics.js` funzioni vuote:
     ```js
     export function addUTM(url, campaign){ /* TODO */ }
     export function predictCTR(text){ /* TODO ML */ }
     ```
   • Importa il file nel bundle principale senza rompere build.

6. **Test di regressione**
   • Esegui `pytest` e `npm run build` (se esiste).  
   • Se falliscono, fai rollback delle sole parti incriminate e logga dettagli.
